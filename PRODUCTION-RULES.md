# Production Rules for Zenvik Technologies Website

## Build Requirement
The website must be compatible with cPanel hosting.

## Preferred Frontend Direction
Use a static/frontend-first build that can be uploaded to cPanel public_html.

Allowed:
- HTML
- CSS
- JavaScript
- React/Vite if final build exports static files

Avoid:
- Server-side-only Next.js features
- Backend logic inside frontend
- Database dependency for marketing website
- Anything that cannot run on normal cPanel hosting

## Quality Standard
The website must be:
- Production-grade
- Fast-loading
- Mobile responsive
- SEO-ready
- Secure
- Professional
- Competitive with modern ICT, hosting, software, and digital agency websites

## Development Method
- Work step by step
- Commit after every safe milestone
- Do not mix main-site and portal-site files
- Main site defines the brand style
- Portal site should visually match the main site

## Animation Requirements
- Use modern, smooth animations
- Keep animations lightweight and fast
- Animations must not slow the website
- Use scroll reveal effects for sections
- Use hover effects for cards and buttons
- Use subtle hero animations
- Avoid heavy video backgrounds
- Respect mobile performance
- Website must remain fully usable without animations
