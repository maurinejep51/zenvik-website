# Zenvik Technologies UI/UX System

## Overall Design Direction
Modern enterprise SaaS design inspired by:
- premium hosting companies
- cloud platforms
- enterprise software brands

Style must feel:
- premium
- fast
- trustworthy
- scalable
- modern

## Layout Style
- Wide clean sections
- Spacious padding
- Modern grid layouts
- Rounded cards
- Glass/subtle gradient effects
- Professional shadows
- Strong typography hierarchy

## Header Behavior
- Sticky transparent header on hero
- Solid background on scroll
- Smooth transitions
- CTA button always visible

## Buttons
Primary Button:
- Primary Blue background
- White text
- Smooth hover animation

Secondary Button:
- Gold/outline style

CTA Buttons:
- Use CTA Orange strategically

## Hero Section
Must include:
- strong headline
- subheadline
- dual CTA buttons
- animated visual/illustration
- trust indicators

## Card Design
- Rounded corners
- Soft shadows
- Hover lift effect
- Smooth transitions

## Animations
Use:
- fade up reveals
- stagger animations
- hover interactions
- smooth scrolling
- animated counters
- subtle floating elements

Avoid:
- excessive motion
- slow rendering
- distracting effects

## Mobile UX
Priority:
- fast loading
- easy navigation
- optimized spacing
- sticky CTA behavior
- touch-friendly design

## Portal Matching Strategy
portal.zenviktechnologies.com must visually match:
- colors
- typography
- buttons
- spacing
- cards
- navigation feel

WHMCS should feel like part of the same SaaS ecosystem.
